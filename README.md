# Tower Defense Project

### **Project Overview**

A robust tower defense game featuring random path generation based on a user's input. The player is able to enter a custom map size and have the game generate a map based on these inputs.

The game features 3 towers the player can place, each having unique ranges, damage, health, and firing speed. 
Likewise, the different enemy types have their own health, speed, and damage.
